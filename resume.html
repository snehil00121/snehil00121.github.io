\section{Engagement Overview}
This document outlines the methodology, tooling, and deliverables I apply during web application vulnerability assessments. It is provided as a reference for the proposed engagement on \textbf{complymarket.cloud} (Azure Kubernetes Service).
\section{Methodology}
\subsection{Phase 1 — Reconnaissance}
\begin{itemize}[leftmargin=*, itemsep=2pt]
    \item Passive fingerprinting: HTTP headers, TLS configuration, server banners
    \item Subdomain and endpoint enumeration via Amass, Subfinder, and ffuf
    \item JavaScript file analysis for exposed API keys, endpoints, and secrets
    \item External AKS surface review: publicly exposed services, ingress rules, and misconfigured endpoints
\end{itemize}
\subsection{Phase 2 — Vulnerability Assessment}
\begin{itemize}[leftmargin=*, itemsep=2pt]
    \item Manual web application testing across the full OWASP Top 10
    \item Authentication and session management analysis
    \item Broken access control and IDOR testing
    \item SQL Injection and XSS — manual verification, not scanner-only
    \item API endpoint enumeration and security misconfiguration review
    \item Container and Kubernetes misconfiguration review using Trivy (external surface only, as scoped)
    \item Automated surface coverage via Nuclei and Burp Suite Pro active scan
\end{itemize}
\subsection{Phase 3 — Reporting}
\begin{itemize}[leftmargin=*, itemsep=2pt]
    \item All findings rated by CVSS v3.1 severity: Critical / High / Medium / Low
    \item Each finding includes: description, proof of concept, business impact, remediation steps
    \item Report written for both technical and non-technical stakeholders
    \item Delivered as a clean PDF within the agreed timeline
\end{itemize}
\section{Tooling}
\begin{itemize}[leftmargin=*, itemsep=2pt]
    \item \textbf{Web Application:} Burp Suite Professional, OWASP ZAP

    \item \textbf{Scanning:} Nuclei, Nmap, Nikto, Nessus

    \item \textbf{Cloud:} Trivy (container/AKS misconfiguration), Cloudsploit
    \item \textbf{Scripting:} Python, Bash for custom automation
\end{itemize}
